Codebook · three rules that separate codebook from wish

Rule 01

Operational

Each rule names a feature you can find in the text. "Mentions Watch List" works. "Has a critical tone" does not.

Rule 02

Disjoint

No paragraph fits two rules. Praise + one concern resolves to exactly one bin.

Rule 03

Testable

Two coders applying the rules to the same paragraph reach the same answer most of the time.

Aggregation choices

Mean
Simple. Treats every paragraph as equally informative.
Length-weighted
Verbose paragraphs count more. Re-read your data first.
Top-K
Keep the K most extreme paragraphs. Validate K against gold.

Hypothesis engineering · the recipe

Start simple. Three hypotheses. One positive, one negative, one for the residual class. Use multi_label=True so each hypothesis returns its own entailment probability. Score formula. score = sum(weight_i × prob_i) / sum(|weight_i|). Add hypotheses in response to gold-set misses. Re-read 10 disagreements. Group by failure mode. For each mode, add one targeted hypothesis. Set weights by codebook judgment, not by tuning. Stop adding hypotheses when kappa stops improving. More than 15 rarely helps.

Code · two snippets to keep

Ensemble · weighted entailment scores
from transformers import pipeline import numpy as np zsc = pipeline("zero-shot-classification", model="MoritzLaurer/DeBERTa-v3-" "base-mnli-fever-anli") hyps = ["text praises target", "text criticizes target"] w = np.array([+1.0, -1.0]) p = zsc(text, hyps, multi_label=True) prob = np.array([ p["scores"][p["labels"].index(h)] for h in hyps]) score = (prob*w).sum() / np.abs(w).sum()
Validation · kappa, correlation, confusion
from sklearn.metrics import ( cohen_kappa_score, confusion_matrix) from scipy.stats import pearsonr k = cohen_kappa_score(gold, pred) r, _ = pearsonr(score, gold) cm = confusion_matrix(gold, pred, labels=[-1,0,1]) print(f"kappa {k:+.3f} r {r:+.3f}") print(cm) # re-read off-diagonals by hand

Pitfalls

  • Skipping the gold set. No gold means no kappa means no defense. Code 30 paragraphs.
  • Tuning weights to the gold. Weights come from the codebook. If you train them, you need a held-out set.
  • Single hypothesis on mixed text. The model picks the dominant signal. Use an ensemble.
  • Accuracy alone. A class-imbalanced corpus produces 80% accuracy from a constant predictor. Always report kappa.
  • Mean aggregation on uneven coverage. A country-year with one paragraph is not the same as one with forty. Always report n.
  • Forgetting subgroups. Headline kappa hides drift. Stratify by region or time period in the appendix.
